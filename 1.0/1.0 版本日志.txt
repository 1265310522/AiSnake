# 作者: Mr丶whale
# 项目名称：贪吃蛇 
 
# 版本号：1.0

# 项目类型: C++

# 项目功能：

  1. 与普通贪吃蛇功能一样（w、s、a、d 或者 方向键控制方向）
  
  2. 游戏开始时，在未按下方向键时蛇静止
  
  3. 游戏具有暂停功能，在暂停过程中，能够更换速度模式（按下 R 或 r 返回自动模式）
  
  4. 蛇的速度分为两个模式：自动模式与手动模式
  
  5. 自动模式下，蛇的速度跟随分数的增加而加快
  
  6. 手动模式下，按 + 加速，按 - 减速 ，在未改变模式情况下蛇的速度恒定为当前所调节的速度
  
  7. 在游戏过程中也能调节速度和更换模式

# 项目缺陷：
 
  1. 此版本不具有存储最高分数功能
  
  2. 由于每次会删除所有蛇身然后再进行打印，所有当蛇身过多时，会有明显的刷新感
  
# 项目说明：
 
  1. 该项目的蛇由 vector 存储（非链表蛇）
  
  2. 该项目的仅有代码与注释说明，无详细过程
